# Rock Paper Scissors Game

A simple console-based Rock Paper Scissors game built in Java where the player competes against the computer.

## Features

- 🎮 Play against the computer
- 🎲 Random computer moves
- ✅ Input validation
- 🔄 Play multiple rounds
- 📊 Live score tracking
- 🏆 Final winner announcement
- ❓ Play Again option

## Technologies Used

- Java
- IntelliJ IDEA

## Java Concepts Practiced

- Variables
- Arrays
- Scanner
- Random
- if-else statements
- do-while loops
- String comparison
- User input validation
