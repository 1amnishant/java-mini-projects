# Slot Machine

A simple console-based Slot Machine game built in Java where the player places bets, spins the reels, and wins rewards based on matching symbols.

## Features

- 🎰 Spin the slot machine
- 💰 Place bets
- 🎲 Randomly generated symbols
- 🏆 Win rewards for matching symbols
- 📊 Balance tracking
- 🔄 Play multiple rounds
- ❓ Play Again option

## Technologies Used

- Java
- IntelliJ IDEA

## Java Concepts Practiced

- Variables
- Arrays
- Scanner
- Random
- Conditional statements (`if-else`)
- Loops (`do-while`)
- User input validation
- Basic game logic
